BoxPtr_t mp4Parser::BoxList::getBox ( const char *  p_boxType) const

Search in the box list which type is equal to parameter p_boxType.

It will only return the first found box even though there are more than one box.If there are more than box with p_boxType, and you need get all the boxes, you can call getNextBox again and again, until the returned box is null.
